## 🛠️ Installation

1. **Clone the Repository**
    ```bash
    git clone https://github.com/spotcache/Classes-in-python.git
    cd Classes-in-python
    ```

2. **Create a Virtual Environment**
    ```bash
    python3 -m venv venv
    ```
